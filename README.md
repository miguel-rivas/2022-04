## Add upstream
```sh
git remote add upstream git@github.com:miguel-rivas/2022.git
```

## Project setup
```sh
yarn install
```

### Vue Cli
```sh
vue ui
```

### Compiles and hot-reloads for development
```sh
yarn serve
```
or
```sh
yarn start
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
